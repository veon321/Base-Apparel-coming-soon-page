# Frontend Mentor - Base Apparel coming soon page solution

This is a clean, responsive, and modern solution to the Base Apparel coming soon page challenge on Frontend Mentor.

## Links

- Solution URL: [https://github.com/veon321/Base-Apparel-coming-soon](https://github.com/veon321/Base-Apparel-coming-soon)
- Live Site URL: [https://veon321.github.io/Base-Apparel-coming-soon-page/](https://veon321.github.io/Base-Apparel-coming-soon-page/)

## Built with

- **Semantic HTML5 markup**: Features distinct semantic blocks including structured form layouts, accessible inputs, and proper heading hierarchies (`<h1>`).
- **CSS Custom Properties (Variables)**: Directly maps standard color tokens and complex linear gradients for background structures and interactive states.
- **Flexbox Layout & Background Layers**: Utilizes flexible containers alongside split-screen background structures to mirror responsive alignments on desktop and mobile displays.
- **Dynamic JavaScript Validation**: Implements real-time input event validation checking for formatting bugs or empty submissions using RegEx matching.

## Features

- **Embedded Input Layout**: Places the submit control asset directly within the bounding frame of the text element using absolute position systems.
- **Dynamic CSS Overlay Transitions**: Leverages color adjustments and minor element scaling transformations during interaction states to offer visual feedback to the client.
